# Preprocesamiento en Ciencia de Datos

## Objetivo
[cite_start]Aplicar de manera práctica el uso de Git y GitHub para la gestión de versiones y la colaboración en proyectos de Ciencia de Datos[cite: 23]. [cite_start]Además, implementar un preprocesamiento completo de datasets usando Pandas, incluyendo la gestión de valores nulos, normalización, codificación de variables categóricas y eliminación de duplicados[cite: 24].

## Estructura básica de carpetas
* `/`: Directorio raíz del proyecto.
* `preprocesamiento.py`: Script de Python donde se implementan las funciones de limpieza y transformación de datos.
* `.gitignore`: Archivo para excluir directorios innecesarios como `.vscode` y `.env`.
* `README.md`: Documentación principal del proyecto.
